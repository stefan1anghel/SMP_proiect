Proiectul nostru consta intr-un robot mobil ce poate transporta obiecte de maxim 500g. Acesta este controlat  prin intermediul unei telecomenzi cu infrarosu.
Robotul se poate deplasa in fata, in spate si poate vira in ambele directii prin antrenajul cu turatie diferita a rotilor motoare.
Componente utilizate:
	- 2 motoare in curent continuu
	- 4 roti
	- Arduino Uno
	- Senzor infrarosu  pentru  telecomanda
	- Driver de motor MX 1919 20101
	- Componente de electronica  (fire, rezistente)
